# MCP: Recent Developments

**Last updated:** 2026-03-20

## Current Landscape

MCP has transitioned from an Anthropic project to an industry-wide standard under the Linux Foundation. Adoption by OpenAI, Google DeepMind, and Microsoft means MCP servers are portable across AI tools. The ecosystem is maturing rapidly with official registries, curated marketplaces, and thousands of community-built servers.

## Entries

### MCP Donated to Linux Foundation
- **What:** In December 2025, Anthropic transferred governance of MCP to the Linux Foundation, making it a vendor-neutral open standard. OpenAI, Google DeepMind, Microsoft, and others are now contributing.
- **Why it matters:** MCP is no longer tied to one company. Servers you build or install work across Claude, ChatGPT, Gemini, and other tools. This is what makes it worth investing in.
- **Added:** 2026-03-20

### Channels Feature (Research Preview)
- **What:** A new MCP server type that connects Claude Code to Telegram and Discord for async notifications and remote control. Requires Claude Code v2.1.80+.
- **Why it matters:** Removes the need to watch a terminal. Claude sends you a message when something important happens, and you can respond from your phone. Especially useful for long-running tasks.
- **Added:** 2026-03-20

### HeyGen MCP Launch
- **What:** HeyGen released an official MCP server for AI video generation. Claude can now create videos with avatars, scripts, and translations within a coding session.
- **Why it matters:** Expands what you can build with Claude Code into multimedia content, not just text and code.
- **Added:** 2026-03-20

### CAD MCP Server
- **What:** MCP server for computer-aided design tools, enabling Claude to work with 3D models and manufacturing specifications.
- **Why it matters:** Demonstrates MCP's reach beyond software into physical product design and engineering workflows.
- **Added:** 2026-03-20

### Figma MCP Livestream (March 31)
- **What:** Anthropic is hosting a live demonstration of the Figma-to-Claude design-to-code pipeline on March 31, 2026.
- **Why it matters:** The Figma MCP workflow is one of the highest-impact MCP use cases. The livestream will likely show best practices and new capabilities.
- **Added:** 2026-03-20
