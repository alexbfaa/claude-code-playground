# MCP: Notable Servers

**Last updated:** 2026-03-20

## Current Landscape

The MCP server ecosystem is large and growing fast (83,000+ stars on the awesome-mcp-servers list). Most popular services now have MCP servers, and new ones appear daily. The entries below highlight servers that are particularly noteworthy for their impact or novelty.

## Entries

### HeyGen MCP
- **What:** MCP server for HeyGen's AI video generation platform. Lets Claude create videos with AI avatars, custom scripts, and multiple languages directly from a coding session.
- **Why it matters:** Extends Claude Code beyond text and code into video content creation. Useful for demo videos, marketing, and explainer content without recording equipment or editing skills.
- **Added:** 2026-03-20

### CAD MCP Server
- **What:** MCP server for computer-aided design (CAD) tools. Enables Claude to create and modify 3D models, adjust dimensions, and work with manufacturing specifications.
- **Why it matters:** Pushes AI-assisted development into hardware and physical product design. Shows that MCP is not limited to software tools.
- **Added:** 2026-03-20

### Figma MCP
- **What:** Official MCP server for Figma. Gives Claude direct access to design files, including layouts, colors, spacing, typography, and component hierarchy.
- **Why it matters:** Enables the "design-to-code" pipeline where a Figma design can be turned into a working application in hours. One of the highest-impact MCP use cases. Anthropic livestream demo scheduled for March 31, 2026.
- **Added:** 2026-03-20

### Claude Code Channels (Telegram/Discord)
- **What:** MCP server that connects Claude Code to Telegram or Discord. Claude sends notifications and receives commands through messaging apps. Requires v2.1.80+.
- **Why it matters:** Changes the interaction model from babysitting a terminal to getting async notifications on your phone. You can monitor and control Claude Code remotely. Currently in research preview.
- **Added:** 2026-03-20
