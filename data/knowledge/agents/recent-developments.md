# Recent Developments in Agents
**Last updated:** 2026-03-20

## Current Landscape
The agent ecosystem around Claude Code is growing fast, with both official features and community-built tools expanding what's possible. Key trends include team-based agent reviews, large public agent libraries, and tooling to manage agents at scale.

## Entries

### Claude Code Review
- **What:** Official Anthropic feature where a team of agents reviews pull requests. Currently in research preview for Team and Enterprise plans, priced at $15-25 per review.
- **Why it matters:** This is the first official "multi-agent product" from Anthropic. It shows the direction of agents as collaborative teams, not just individual assistants. If it expands beyond Team/Enterprise, it could change how solo developers get code reviewed.
- **Added:** 2026-03-20

### Claude Code Templates
- **What:** Hackathon-winning project that bundles agents, skills, hooks, and settings into shareable starter kits. Includes visual dashboards for monitoring agent activity.
- **Why it matters:** Lowers the barrier to entry. Instead of building agents from scratch, you can start from a proven template and customize. The visual dashboards are notable because they address the "what is my agent doing?" visibility problem.
- **Added:** 2026-03-20

### Non-Engineering Agent Use Cases
- **What:** People are building Claude Code agents for tasks beyond software engineering. Examples include DTC (direct-to-consumer) brand management agents and Google Maps lead generation agents.
- **Why it matters:** Shows that the agent framework is flexible enough for business operations, not just coding. If you run a business, your agents don't have to be about code -- they can handle research, outreach, data processing, and more.
- **Added:** 2026-03-20
