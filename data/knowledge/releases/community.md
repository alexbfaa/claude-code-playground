# Releases: Community

**Last updated:** 2026-03-20

## Current Landscape

Anthropic is investing heavily in community infrastructure around Claude Code. Free courses, conferences, ambassador programs, and official skill/server repositories are building an ecosystem beyond the core product. The community is large and active, with open-source repositories reaching tens of thousands of stars.

## Entries

### Anthropic Academy
- **What:** Six free courses covering Claude fundamentals, prompt engineering, Claude Code workflows, agent design, and advanced techniques. Available at academy.anthropic.com.
- **Why it matters:** Structured learning path for all skill levels. Especially useful for non-technical users who want to get more out of Claude Code without relying on trial and error.
- **Added:** 2026-03-20

### Code with Claude Conference
- **What:** Anthropic's developer conference scheduled for spring 2026, with events in San Francisco, London, and Tokyo. Focused on Claude Code, agents, and AI-assisted development.
- **Why it matters:** First major in-person event focused on Claude Code. Likely to include announcements, workshops, and networking with other Claude Code users and the Anthropic team.
- **Added:** 2026-03-20

### Community Ambassadors Program
- **What:** Anthropic's program recognizing active community members who help others learn Claude Code. Ambassadors get early access to features and direct communication with the product team.
- **Why it matters:** Creates a feedback loop between power users and Anthropic's development team. Ambassador insights often influence product direction.
- **Added:** 2026-03-20

### anthropics/skills Repository
- **What:** Anthropic's official open-source repository of Claude Code skills. Contains production-ready skills for common tasks. Has 87,000+ GitHub stars.
- **Why it matters:** The primary source for high-quality, vetted skills. Instead of building skills from scratch, you can find and adapt existing ones. The star count reflects massive community interest and adoption.
- **Added:** 2026-03-20

### awesome-mcp-servers Repository
- **What:** Community-maintained list of MCP servers with 83,000+ GitHub stars. Categorized by use case with installation instructions and quality ratings.
- **Why it matters:** The go-to directory for finding MCP servers. Before building a custom server, checking this list first can save significant time. The high star count indicates it is well-maintained and widely trusted.
- **Added:** 2026-03-20
