# Skills Ecosystem
**Last updated:** 2026-03-20

## Current Landscape
The skills ecosystem spans a 65,000+ item marketplace, an official GitHub repository with 87K+ stars, an open standard adopted by four major AI tools, and distribution channels that reach beyond developers into Office apps and non-technical workflows.

## Entries

### Community Marketplace (65,000+ Skills)
- **What:** A community-driven directory listing skills across many categories: development workflows, marketing, business strategy, API integrations, testing, and more.
- **Why it matters:** The ecosystem has critical mass. Before building a skill from scratch, check the marketplace -- someone likely built something similar you can adapt.
- **Added:** 2026-03-20

### anthropics/skills GitHub Repository
- **What:** The official Anthropic-maintained skills repository on GitHub with 87,000+ stars. Serves as the central hub for sharing and discovering portable skills.
- **Why it matters:** The canonical source for high-quality, community-vetted skills. Also the reference implementation for the SKILL.md format.
- **Added:** 2026-03-20

### SKILL.md Open Standard Adoption
- **What:** The SKILL.md format, originally created for Claude Code, was adopted in Dec 2025 by OpenAI (Codex CLI, ChatGPT), Cursor, and Gemini CLI (Google).
- **Why it matters:** Write a skill once, use it everywhere. Skills are portable across AI coding tools, which protects your investment in building workflows regardless of which tool you or your team uses.
- **Added:** 2026-03-20

### Notable Skill Packs
- **What:** Bundled skill collections for specific use cases. Key examples: YC CEO pack (8 skills for executive strategy by Garry Tan), DTC brand pack (marketing workflows by Mike Futia), Uber iMessage skill (enterprise open-source example).
- **Why it matters:** Skill packs are the primary distribution format for teams. They let you install a full set of workflows in one step instead of finding and configuring skills individually.
- **Added:** 2026-03-20

### Office Add-ins (Excel, PowerPoint)
- **What:** Claude skills available as Microsoft Office add-ins. Skills appear as buttons in the Office sidebar. Teams build skills; non-technical colleagues run them with one click from their spreadsheets or presentations.
- **Why it matters:** Extends the skills ecosystem to people who don't use the command line. A team's developer can build a skill, and the marketing or finance team can run it from Excel without any setup.
- **Added:** 2026-03-20
